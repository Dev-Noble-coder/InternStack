# 🚀 InternStack

> **The ultimate SIWES engine for students in Nigeria.**

InternStack is a modern web platform designed to streamline the Students Industrial Work Experience Scheme (SIWES). Built to help students effortlessly generate professional CVs, master and manage their logbooks, and discover tailored SIWES opportunities from top Nigerian companies.

## 🌟 Key Features

- **Professional CV Generation**: Quickly build a standout resume to land top internships.
- **Logbook Mastery**: Easily track and manage your daily activities and SIWES records.
- **Opportunity Discovery**: Access curated SIWES slots and internships from leading companies across Nigeria.
- **Modern & Dynamic UI/UX**: Enjoy a fast, responsive, and beautiful interface with smooth animations and layout transitions.

## 🛠️ Tech Stack

This project is built using modern web technologies to ensure high performance and an exceptional user experience:

- **Framework:** [Next.js 16](https://nextjs.org/)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

Follow these simple steps to start developing locally.

### Prerequisites

Ensure you have Node.js (v20+ recommended) and `npm` installed.

### Installation & Setup

1. **Clone the repository:**
   ```sh
   git clone https://github.com/yourusername/intern-stack.git
   cd intern-stack
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   ```

4. **View the Application:**
   Open your browser and navigate to [http://localhost:3000](http://localhost:3000).

## 📂 Core Project Structure

- `app/` - The main Next.js App Router structure.
- `app/components/` - Key UI components including `NavBar`, `HeroSection`, `ProblemStatement`, `Solution`, and `Footer`.
- `public/` - Static assets (e.g., logos, fonts).
